# Multi Board Flipper Zero

## what this board contains:
- NRF24 (with decoupling capacitors)
- ESP32
- Sentry safe pins
- Power LED

## components:
* D1: 2V LED 
* R1: 68 R
* C1: 0.1 uF
* C2: 10 uF

### all components are smd 1206
